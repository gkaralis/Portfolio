Project overview

Microsoft Edge launched in 2015 as the web browser for Windows 10, unseating Internet Explorer as the default browser. Redesigning such an essential app for users & the business from the ground up was an incredible opportunity. Replacing a 20-year-old product in a tiny fraction of that time was a major feat. Some experiences landed smoothly; others felt some V1 pain.

A major usability issue was identified late in the release, where users upgrading from previous versions of Windows did not understand how to import their favorites from other browsers. I was tasked with developing a "first run experience" to address the usability problem, along with promote other product goals.
